The ktichensink includes every single element in the framework. The kitchensink is typically used for visual regression testing. After you change one element, see if it doesn't inflict another element. But also the other way around, if you change a certain element, sometimes other related elements need to change as well to maintain the visual coherence between the elements.

We would advise not to use this as a component library. Instead check the element's page itself in the catalog. This will contain info about how to use and implementation details.
